Detailed Description

template<class... Comps>
class Zipper< Comps >

Range adapter that simultaneously iterates over multiple Comps.

Template Parameters
CompsVariadic template parameter pack of Comp types to zip together.

This class provides a convenient interface for parallel iteration over multiple Comps. It returns a ZipperIterator that yields tuples of values from each Comp where all Comps have valid (non-empty optional) values at the same position.

Note
The zipper respects the size of the smallest Comp to prevent out-of-bounds access.
Only positions where all Comps have valid values are yielded during iteration.
SparseArray<Position> positions;
SparseArray<Velocity> velocities;
for (auto [pos, vel] : Zipper(positions, velocities)) {
pos.x += vel.dx;
pos.y += vel.dy;
}
