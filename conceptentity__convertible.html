Detailed Description

Checks if a type has a change_entity method for mapping entity IDs.

Requires a change_entity method that maps entities using a TwoWayMap. The method must be const, return the same type, and accept a TwoWayMap<size_t, size_t> (compatible with Registry::Entity). Use this for events containing Registry::Entity fields.

struct MyEvent {
Registry::Entity actor;
auto change_entity(TwoWayMap<Registry::Entity, Registry::Entity> const&)
const -> MyEvent;
};
static_assert(EventHasChangeEntity<MyEvent>);
