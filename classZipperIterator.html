Detailed Description

template<class... Comps>
class ZipperIterator< Comps >

Iterator that simultaneously traverses multiple Comps and yields tuples of their elements.

Template Parameters
CompsVariadic template parameter pack of Comp types to iterate over.

This iterator implements a zipper pattern that combines multiple Comps into a single iteration stream. It only yields values where all Comps have valid (non-empty optional) values at the current position. The iterator automatically skips positions where any Comp lacks a value.

Note
The iterator uses const iterators (Cit) for const Comps and regular iterators (It) for non-const Comps.
Iteration stops at the size of the smallest Comp to prevent out-of-bounds access.
