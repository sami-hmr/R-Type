Detailed Description

RGBA color with 8-bit channels.

Represents colors using four unsigned char components (0-255):

  • r: Red channel
  • g: Green channel
  • b: Blue channel
  • a: Alpha channel (transparency, 255 = fully opaque)

Supports JSON parsing with optional hook system for dynamic color binding. Provides predefined color constants (WHITE, BLACK, RED, GREEN, BLUE, TRANSPARENT).

See also
colorToByte(), parseColor()
